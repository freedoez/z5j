# 1德摩根律
$\overline{A \cup B} = \overline{A} \cap \overline{B}$
$\overline{A \cap B} = \overline{A} \cup \overline{B} = \overline{A} \cdot \overline{B}$
记忆口诀：长杠变短杠，开口变方向

# 2概率的性质
1.非负性：$0 \le P(A) \le 1$
2.规范性：$P(\Omega)=1,\ P(\emptyset)=0$
3.有限可加性：若A，B互斥，则$P(A\cup B)=P(A)+P(B)$ ，($AB=\emptyset$)
4.互补性：$P(\overline{A})=1-P(A)$

$A \cup B \cup C \Leftrightarrow A, B, C \text{至少发生一个}$

# 3条件概率
设 $A,B$ 是两个事件，且 $P(A) > 0$，称 $P(B|A) = \frac{P(AB)}{P(A)}$ 为事件 $B$ 在事件 $A$ 条件下的条件概率。
