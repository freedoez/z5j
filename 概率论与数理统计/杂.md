# 德摩根律
$\overline{A \cup B} = \overline{A} \cap \overline{B}$
$\overline{A \cap B} = \overline{A} \cup \overline{B} = \overline{A} \cdot \overline{B}$
记忆口诀：长杠变短杠，开口变方向

# 概率的性质
1.非负性：$0 \le P(A) \le 1$
2.规范性：$P(\Omega)=1,\ P(\emptyset)=0$
3.有限可加性：若A，B互斥，则$P(A\cup B)=P(A)+P(B)$ ，($AB=\emptyset$)
4.互补性：$P(\overline{A})=1-P(A)$

$A \cup B \cup C \Leftrightarrow A, B, C \text{至少发生一个}$
# 古典、几何概型

## 古典
 1. 有限性：试验 E 的样本空间$\Omega$ 是有限集合，即$\Omega = \{e_1, e_2, \dots, e_n\}$

2. 等可能性：每个基本事件发生的概率相等，即$P(e_1) = P(e_2) = \dots = P(e_n) = \frac{1}{n}$

## 几何
1. 样本空间与几何区域相关：
试验的样本空间$\Omega$对应一个可度量的几何区域，这个区域可以是一维的线段、二维的平面区域，也可以是三维的立体空间。
2. 等可能性：
向样本空间$\Omega$对应的几何区域内任意投一点，该点落在区域内任何一点的可能性是相等的。

- 其核心概率公式为：

$P(A) = \frac{\text{事件}A\text{对应的区域测度}}{\text{样本空间}\Omega\text{对应的区域测度}}$    
<small><small>“测度”在一维情况下是长度，二维是面积，三维是体积。</small></small>

# 条件概率
设 $A,B$ 是两个事件，且 $P(A) > 0$，称 $P(B|A) = \frac{P(AB)}{P(A)}$ 为事件 $B$ 在事件 $A$ 条件下的条件概率。
注：
① $P(\Omega|A) = 1$；② $P(\emptyset|A) = 0$
③$P(\overline{B}|A) = 1 - P(B|A)$![](assets/杂/file-20260212202722576.png)
④$P(B \cup C|A) = P(B|A) + P(C|A) - P(BC|A)$
⑤$P(B - C|A) = P(B|A) - P(BC|A)$
<small>对④⑤:抹去A，算完后再补上A</small>


# 事件的独立性
1. 事件的独立 
   若两个事件 $A,B$ 满足 $P(B|A) = P(B)$，则称事件 $A$ 与 $B$ 相互独立。
<span class="small-note"> 
   注：
   ① 独立的本质：$A$ 与 $B$ 互不影响。
   ② 独立的等价定义： $A$ 与 $B$ 独立 $\iff P(AB) = P(A)P(B) \iff P(B|A) = P(B|\overline{A})$ $\iff P(B|\overline{A}) = P(B) \iff P(B|A) + P(\overline{B}|\overline{A}) = 1$ 
   ③ 特别地，$\Omega$ 与任意事件 $B$ 独立，$\emptyset$ 与任意事件 $B$ 独立。</span>
