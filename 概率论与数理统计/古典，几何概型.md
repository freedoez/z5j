# 古典
 
1. 有限性：试验 E 的样本空间$\Omega$ 是有限集合，即$\Omega = \{e_1, e_2, \dots, e_n\}$

2. 等可能性：每个基本事件发生的概率相等，即$P(e_1) = P(e_2) = \dots = P(e_n) = \frac{1}{n}$

# 几何

1. 样本空间与几何区域相关：
试验的样本空间$\Omega$对应一个可度量的几何区域，这个区域可以是一维的线段、二维的平面区域，也可以是三维的立体空间。
2. 等可能性：
向样本空间$\Omega$对应的几何区域内任意投一点，该点落在区域内任何一点的可能性是相等的。

- 其核心概率公式为：

$P(A) = \frac{\text{事件}A\text{对应的区域测度}}{\text{样本空间}\Omega\text{对应的区域测度}}$    
<small><small>“测度”在一维情况下是长度，二维是面积，三维是体积。</small></small>
