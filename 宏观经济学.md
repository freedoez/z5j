
### **一、宏观经济的基本指标及其衡量**

1. **GDP的概念**： 国内生产总值（GDP）是指**一个国家或地区在一定时期内（通常为一年）所生产的全部最终产品与服务的市场价值总和**。理解要点：地域概念（国土范围内）、流量概念（一定时期）、最终产品（避免重复计算）、市场价值（按市场价格计算）。
    
2. **GDP的核算方法（重点支出法）**： 支出法从最终产品的流向角度进行核算，公式为：
    
    **GDP = 消费（C）+ 投资（I）+ 政府购买（G）+ 净出口（NX）**
    
    - **C**： 居民家庭消费支出。
        
    - **I**： 企业投资（固定资产投资+存货投资）。注意：不包括购买股票等金融投资。
        
    - **G**： 政府购买产品与服务的支出。注意：不包括转移支付（如养老金、失业救济金）。
        
    - **NX**： 出口（X） - 进口（M）。
        
    
3. **衡量宏观经济的3个核心指标**：
    
    - **国内生产总值（GDP）**： 衡量经济总量和经济增长速度。
        
    - **失业率**： 衡量劳动力资源的利用程度，反映社会稳定性。
        
    - **通货膨胀率**： 衡量物价总水平的变动幅度，反映货币购买力的稳定性。
        
    
4. **储蓄一投资恒等式（重点四部门）**： 这是基于会计定义、事后的恒等关系。
    
    **四部门经济（包含家庭、企业、政府、国外部门）的恒等式为：**
    
    `I = S + (T - G) + (M - X)`
    
    - **I**： 投资
        
    - **S**： 私人储蓄
        
    - **T - G**： 政府储蓄（政府税收减去政府支出，即预算盈余）
        
    - **M - X**： 国外对本国的储蓄（本国进口大于出口，相当于外国对本国的净投资）

### **二、简单国民收入决定理论**

1. **边际消费倾向（MPC）**： 指**增加一单位收入中用于增加消费的部分所占的比率**。公式： MPC = ΔC / ΔY。**取值范围**： 0 < MPC < 1。
    
2. **边际储蓄倾向（MPS）**： 指**增加一单位收入中用于增加储蓄的部分所占的比率**。公式： MPS = ΔS / ΔY。**与MPC的关系**： **MPC + MPS = 1**。
    
3. **乘数**：
    
    - **概念**： 指**自发性支出的变动所引起的国民收入变动的倍数**。
        
    - **四种乘数表达式（以三部门经济为例，定量税T0）**：
        
        - 政府购买支出乘数 (Kg)： `Kg = 1 / (1 - MPC)`
            
        - 税收乘数 (Kt)： `Kt = -MPC / (1 - MPC)`
            
        - 政府转移支付乘数 (Ktr)： `Ktr = MPC / (1 - MPC)`
            
        - 平衡预算乘数 (Kb)： 当政府购买和税收增加同等数量时，`Kb = 1`
            
        
    - **大小关系**： **Kg > Ktr > Kb > Kt**（绝对值比较）。**重点比较**： |Kg| > |Ktr|，因为政府购买直接是总支出的一部分，而转移支付需先转化为个人可支配收入，再按MPC进行消费。
        
    - **计算题要点**： 区分税收是定量税还是比例税。比例税下，乘数分母变为 `1 - MPC(1 - t)`，其中t为税率，乘数效应会减小。

### **三、产品市场和货币市场的一般均衡（IS-LM模型）**

1. **IS曲线的意义**： 描述**产品市场达到均衡（即投资I等于储蓄S）时，国民收入（Y）与利率（r）之间反向变动关系**的曲线。
    
    - **IS曲线左边的点**： 表示产品市场存在**过度需求**（I > S）。
        
    - **IS曲线右边的点**： 表示产品市场存在**过度供给**（I < S）。
        
    
2. **IS曲线的移动**： **扩张性财政政策（增加G，减T，增加TR）会使IS曲线向右上方移动**；紧缩性财政政策则相反。任何能增加总支出的因素（如消费者信心增强导致C增加，企业家乐观导致I增加）都会使IS曲线右移。
    
3. **LM曲线的意义**： 描述**货币市场达到均衡（即货币需求L等于货币供给M）时，国民收入（Y）与利率（r）之间同向变动关系**的曲线。
    
    - **LM曲线左边的点**： 表示货币市场存在**过度货币供给**（L < M）。
        
    - **LM曲线右边的点**： 表示货币市场存在**过度货币需求**（L > M）。
        
    - **LM曲线的移动**： **扩张性货币政策（增加货币供给M）会使LM曲线向右下方移动**；紧缩性货币政策则相反。
        
    
4. **IS-LM的计算题**： 联立IS方程（Y = C + I + G + NX）和LM方程（M/P = L(Y, r)）求解均衡收入Y和均衡利率r。
    
5. **画图分析**：
    
    - **财政政策**： 增加G → IS曲线右移 → 新均衡点：Y增加，r上升（存在“挤出效应”）。
        
    - **货币政策**： 增加M → LM曲线右移 → 新均衡点：Y增加，r下降。

### **四、宏观经济政策分析与实践**

1. **宏观经济政策目标**： ==**充分就业、物价稳定、经济增长、国际收支平衡**==。
    
2. **财政政策**：
    
    - **扩张性财政政策**： 在经济萧条时采用，旨在刺激总需求。措施：**增加政府购买（G）、减税（T）、增加转移支付（TR）**。
        
    - **紧缩性财政政策**： 在经济过热、通胀严重时采用，旨在抑制总需求。措施：减少G、增税、减少TR。
        
    
3. **货币政策**：
    
    - **扩张性货币政策**： 在经济萧条时采用，旨在刺激总需求。措施：**降低法定准备金率、降低再贴现率、在公开市场上买入债券（增加货币供给）**。
        
    - **紧缩性货币政策**： 在经济过热时采用，旨在抑制总需求。措施：提高法定准备金率、提高再贴现率、在公开市场上卖出债券（减少货币供给）。
        
    
4. **论述题要点**：
    
    - **识别政策**： 先判断题目中的措施属于财政政策（涉及政府收支）还是货币政策（涉及央行和货币量）。
        
    - **模型分析**：
        
        - **用IS-LM模型**： 分析政策对均衡收入和利率的影响。例如，扩张性财政政策使IS右移，可能伴随货币政策的调整。
            
        - **用AD-AS模型**： 分析政策对总需求曲线（AD）的影响，进而影响物价水平（P）和总产出（Y）。扩张性政策使AD曲线右移，在短期会提高Y和P。
            
        
    - **扩展分析**： 考虑政策的局限性（如财政政策的挤出效应、时滞；货币政策的流动性陷阱）和组合使用（如宽松货币配合扩张财政以减少挤出效应）。

### **五、AD-AS模型**

1. **AD曲线向右下方倾斜的原因**：
    
    - **财富效应（庇古效应）**： P下降 → 实际货币余额增加 → 人们感到更富有 → 消费（C）增加 → Y增加。
        
    - **利率效应（凯恩斯效应）**： P下降 → 实际货币供给(M/P)增加 → 利率r下降 → 投资（I）增加 → Y增加。
        
    - **汇率效应（蒙代尔-弗莱明效应）**： P下降 → 利率r下降 → 资本外流 → 本币贬值 → 净出口（NX）增加 → Y增加。
        
    
2. **AS曲线的几种情况及长期总供给曲线垂直的原因**：
    
    - **短期总供给曲线（SRAS）**： 向右上方倾斜（由于工资、价格粘性）。
        
    - **长期总供给曲线（LRAS）**： **垂直**。原因：在长期，价格和工资具有完全弹性，经济总会恢复到**充分就业的产出水平（潜在GDP）**。此时产出由经济的技术、资源禀赋（资本、劳动）决定，与物价水平无关。
        
    
3. **导致短期总供给曲线移动的因素**： 主要是**生产成本**的变化。
    
    - 原材料价格（如石油价格）上涨 → SRAS左移。
        
    - 名义工资水平普遍上涨 → SRAS左移。
        
    - 技术进步 → SRAS右移。
        
    - 预期通货膨胀率上升 → 工人要求更高工资 → SRAS左移。

### **六、失业与通胀**

1. **失业的分类、原因、对策（论述题）**：
    
    - **摩擦性失业**： 因劳动力正常流动（换工作、找新工作）产生。原因：信息不对称。对策：完善就业信息服务。
        
    - **结构性失业**： 因经济结构变化（如产业升级）导致劳动力技能与岗位不匹配。原因：技术革新、消费结构变化。对策：提供职业培训、教育补贴。
        
    - **周期性失业**： 因经济周期中的衰退期总需求不足产生。原因：总需求不足。对策：实施扩张性的财政和货币政策。
        
    
2. **自然失业率的概念**： **自然失业率 = 摩擦性失业率 + 结构性失业率**。这是经济在**充分就业**状态下仍然存在的失业率，**不是100%就业**。
    
3. **短期菲利普斯曲线的定义**： 描述**失业率与通货膨胀率之间在短期内存在替代关系**的曲线。失业率低，通胀率高；失业率高，通胀率低。
    
4. **长期菲利普斯曲线的形状**： **长期菲利普斯曲线是一条垂直于自然失业率的直线**。表明在长期，由于人们的通胀预期调整，失业率与通胀率之间不存在替代关系。
    
5. **滞胀是需求拉动导致还是成本推动导致？**： **成本推动**。当总供给曲线向左移动时（如石油危机），会导致物价水平上涨（通胀）和产出下降（失业增加），即“滞胀”。
    
6. **需求拉动与成本推动的通胀（画图分析）**：
    
    - **需求拉动的通胀**：
        
        - **原因**： 总需求过度增长（AD曲线右移）超过潜在产出能力。
            
        - **对策**： 采取紧缩性的财政和货币政策（使AD曲线左移回原位）。
            
        - **图形**： AD曲线右移，与垂直的LRAS相交于更高的价格水平。
            
        
    - **成本推动的通胀**：
        
        - **原因**： 生产成本上升（如工资、油价上涨）导致SRAS曲线左移。
            
        - **对策**： 非常棘手。紧缩政策会加剧失业，宽松政策会加剧通胀。通常建议通过政策改善供给（如补贴、减税）使SRAS曲线右移回原位。
            
        - **图形**： SRAS曲线左移，导致物价上涨，产出下降。

### **七、开放经济下的宏观经济模型**

1. **BP的概念**： BP曲线表示**国际收支平衡**。BP = 0 意味着**经常项目余额（NX）与资本项目余额（F）之和为零**，即 `BP = NX + F = 0`。
    
2. **BP曲线移动的因素**：
    
    - **BP曲线右移**： 本币贬值、外国利率上升、世界需求增加（使NX增加，更易实现平衡）。
        
    - **BP曲线左移**： 本币升值、本国利率上升（吸引资本流入，更易实现平衡）。
        
    
3. **IS*-LM*模型中政策的有效性（蒙代尔-弗莱明模型）**：
    
    - **浮动汇率制**：
        
        - **财政政策无效**： 扩张性财政政策 → 利率有上升压力 → 资本流入 → 本币升值 → 净出口减少 → 完全挤出效应，总收入不变。
            
        - **货币政策有效**： 扩张性货币政策 → 利率下降 → 资本流出 → 本币贬值 → 净出口增加 → 总收入显著增加。
            
        
    - **固定汇率制**：
        
        - **财政政策有效**： 扩张性财政政策 → 利率有上升压力 → 资本流入 → 本币有升值压力 → 央行为了维持汇率必须买入外币、卖出本币 → 货币供给自动增加，强化了财政政策效果。
            
        - **货币政策无效**： 扩张性货币政策 → 利率下降 → 资本流出 → 本币有贬值压力 → 央行为了维持汇率必须卖出外币、买入本币 → 货币供给自动减少，政策效果被抵消。
            
        
    
4. **三元悖论**： 一个国家在**资本自由流动、货币政策独立性和固定汇率制**这三个目标中，**最多只能同时实现两个**，不可能三者兼得。

### **八、经济增长和经济周期理论**

1. **经济增长与经济发展的联系和区别**：
    
    - **经济增长**： 是一个**数量**概念，主要指GDP或人均GDP的增加。
        
    - **经济发展**： 是一个更广泛的**质量**概念，不仅包括经济增长，还包括经济结构的优化、生活质量的提高、社会制度的完善等。
        
    
2. **稳态（Solow模型）**： **指人均资本存量（k）和人均产出（y）不再发生变化的状态**。
    
    - **稳态条件**： **投资（sf(k)）正好等于资本折旧和人口增长所消耗的资本量（(δ+n)k）**，即 `sf(k) = (δ+n)k`。
        
    
3. **资本的黄金律水平**： **指使稳态时人均消费达到最大化的那个人均资本存量水平**。
    
    - **黄金律条件**： **资本的边际产量（MPK）等于折旧率与人口增长率之和**，即 `MPK = δ + n`。在黄金律水平，社会福利最大化。
